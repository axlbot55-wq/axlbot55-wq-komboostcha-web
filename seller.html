Google Sheets hookup

Use the included Apps Script as the backend target.

1. Open backend/google-apps-script/Code.gs
2. Paste it into a Google Apps Script project bound to your sheet
3. Deploy as a web app with access for anyone with the link
4. Copy that URL into config.js → orderWebhookUrl
5. New orders will still go to WhatsApp and also log into Sheets automatically
